# `xmerl_ext`

## Description

Set of extensions for Erlang's `xmerl` library (`CDATA`, *XML entities*, *DTD* definitions).

## Development

### Building

```bash
$ rebar3 compile
```

### Testing

```bash
$ rebar3 dialyzer
$ rebar3 proper
```
